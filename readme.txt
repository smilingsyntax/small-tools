== Description ==

Small Tools is a comprehensive WordPress plugin that combines multiple essential features into one lightweight package. It helps optimize your WordPress site's performance, enhance security, and provide useful utilities without the need to install multiple plugins.

= Key Features =

**WordPress General Features:**

* Remove Image Threshold - Allow original size image uploads
* Remove Default Lazy Load - Disable WordPress' built-in lazy loading
* Disable Right-Click - Basic content protection
* Disable Emojis - Remove WordPress emoji scripts
* Remove jQuery Migrate - Clean up script loading
* Custom Admin Footer Text - Customize admin panel footer
* Dark Mode for Admin Panel - Eye-friendly dark theme

**Security Features:**

* Force Strong Passwords - Enhance user account security
* Disable XML-RPC - Prevent potential vulnerabilities
* Hide WordPress Version - Remove version information

**WooCommerce Features:**

* Increase Variation Threshold - Customize WooCommerce variation limits

**Database Optimization:**

* Clean Post Revisions
* Remove Auto Drafts
* Clear Trash Posts
* Delete Spam Comments
* Clean Expired Transients
* Database Table Optimization

**Settings Management:**

* Export Settings - Backup your configuration
* Import Settings - Restore or migrate settings
* User-friendly Interface

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/small-tools` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Access settings via 'Small Tools' in the admin menu

== Frequently Asked Questions ==

= Is this plugin compatible with the latest WordPress version? =

Yes, Small Tools is regularly tested with the latest WordPress versions to ensure compatibility.

= Will this plugin slow down my website? =

No, Small Tools is designed to be lightweight and actually helps improve site performance by providing optimization options.

= Does this work with WooCommerce? =

Yes, Small Tools includes specific features for WooCommerce and checks for WooCommerce compatibility before enabling those features.
